PROYECTO 01

Dashboard de ventas en Excel

Descarga un dataset de ventas de Kaggle, límpialo y crea un dashboard con tablas dinámicas, gráficos y KPIs (ventas por mes, top productos, margen bruto).

Análisis SQL con Northwind DB

Primer análisis con Python

Analiza el dataset Titanic con pandas en Google Colab. Limpia nulos, calcula estadísticas y crea 3 gráficas que cuenten una historia clara con los datos.
